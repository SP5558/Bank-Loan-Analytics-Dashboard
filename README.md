# Bank Loan Analytics Dashboard

## Project Overview
This project is a Power BI dashboard developed to analyze bank loan data and generate business insights related to loan applications, funded amounts, repayments, and customer behavior.
The dashboard helps identify:
- Good vs Bad Loans
- Monthly Loan Trends
- Funded Amount vs Amount Received
- Loan Purpose Analysis
- Interest Rate Analysis
- State-wise Loan Distribution
---
## Tools & Technologies Used
- Power BI
- SQL
- Excel / CSV
- Data Cleaning
- Data Visualization
---
## Dashboard Features
### KPI Metrics
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio
### Visualizations
- Monthly Loan Applications
- Loan Status Analysis
- Home Ownership Analysis
- Loan Purpose Breakdown
- State-wise Loan Analysis
---
## Key Insights
- Good Loans accounted for 83.33% of total applications.
- Total Funded Amount reached 435.8M.
- Debt Consolidation and Credit Card loans were among the major categories.
- Monthly loan applications showed stable growth trends.
